Música Incrustada desde Spotify

Spotify ofrece la posibilidad de incrustar listas de reproducción, álbumes o canciones individuales en tu página web utilizando su reproductor. Algunos de los parámetros/atributos que puedes usar son:

  • src: Es el enlace de la lista de reproducción, álbum o canción que deseas incrustar.
  • width: Define el ancho del reproductor.
  • height: Define la altura del reproductor.
  • frameBorder: Este atributo indica si se debe mostrar un borde alrededor del iframe. El valor 0 significa que no se mostrará ningún borde.
  • allowfullscreen: Permite al iframe entrar en modo de pantalla completa cuando se activa.
  • allow: Este atributo controla qué acciones y capacidades están permitidas para el recurso incrustado. En este caso, se permiten la reproducción automática, la escritura en el portapapeles, el acceso a contenido multimedia cifrado, el modo de pantalla completa y la reproducción en imagen dentro de imagen.
  • loading: Controla cómo se carga el contenido del iframe.
